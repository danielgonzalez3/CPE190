# Library LTC
## Open Source C++ library for SMPTE TimeCode LTC Protocol

Supported board :

-	Orange Pi Zero

Supported input :

- LTC SMPTE
- TCNet
- Art-Net
- rtpMIDI
- MIDI
- System Time

Supported output :

* LTC SMPTE
* Art-Net
* MIDI
* rtpMIDI
* Displays :   
   * OLED display 128 x 64 I2C (auto-detect)
      * SSD1306 I2C
      * SH1106 I2C 
   * LCD I2C (auto-detect)
      * 16x2 
      * 16x4
   * MAX7219 (SPI)
      *  8x 7-segment 
      *  Matrix
   * WS28xx (SPI)
      *  7-segment  
      *  Matrix 64 x 8 (1 line)
   * HUB75 (GPIO)
       * Matrix 64 x 32 (4 lines)
